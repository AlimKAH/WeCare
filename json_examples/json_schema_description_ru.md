# Описание схемы данных WeCare

## Продукт (product)
- **id**: Уникальный идентификатор продукта
- **barcode**: Штрих-код продукта
- **name**: Название продукта
- **manufacturer**: Производитель
- **weight**: Вес/объем продукта
  - **value**: Числовое значение
  - **unit**: Единица измерения (г, мл)

## Ингредиенты (ingredients)
Массив ингредиентов продукта:
- **name**: Название ингредиента
- **safety**: Безопасность ("safe", "suspicious", "harmful")

## Пищевая ценность (nutrition)
- **calories**: Калории на 100г
- **protein**: Белки на 100г
- **fat**: Жиры
  - **total**: Общее содержание жиров
  - **saturated**: Насыщенные жиры
- **carbohydrates**: Углеводы
  - **total**: Общее содержание углеводов
  - **sugar**: Сахар
- **fiber**: Клетчатка
- **salt**: Соль
- **sodium**: Натрий

## Оценка (score)
- **total**: Общая оценка продукта (0-100)
- **category**: Категория качества ("Excellent", "Good", "Average", "Low Quality", "Very Low Quality")
- **nutrition_score**: Оценка пищевой ценности
- **additives_score**: Оценка по добавкам

## Добавки и изображение
- **additives**: Массив пищевых добавок (E-коды)
- **image_url**: URL изображения продукта

## Анализ аллергенов (allergens_analysis)
- **detected_allergens**: Обнаруженные аллергены в продукте
- **user_allergens_present**: Аллергены, опасные для пользователя

## Совместимость с диетами (diet_compatibility)
Массив диет с информацией о совместимости:
- **diet**: Название диеты
- **compatible**: Совместимость (true/false)
- **reason**: Причина совместимости/несовместимости

## Дополнительно
- **scan_timestamp**: Временная метка сканирования
